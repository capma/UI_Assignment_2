Chapter 06: Designing navigation

« Previous Next »

Web navigation conventions will help users know what to look for and where to look when they need them.

Should have global navigation - a set of navigation elements that appear in the same place on every page.


Global navigation (or persistent navigation) should include these 5 elements at all times:

  1. The Site ID (logo) on the top left of the page

  2. A way home on the top right of the page

  3. A way to search on the top right, next to a way home

  4. Primary navigation, that is the links to the main sections of the site

  5. Utilities (shopping cart, site map, help etc.)


The only exception is sometimes the homepage doesn't need the persistent navigation because it is not like other pages, and certain forms pages (e.g. when I'm paying for my purchases on an e-commerce site I don't want to do anything except finish filling in the forms).


There are 4 things you should know about page names:

  • Every page needs a page name.

  • The page name needs to be in the right place: framing the content that is unique to this page.

  • It needs to be noticable. It should say: "This is the heading for the entire page".

  • Most important, the page name needs to match what users clicked. If users click on a link that says "Women sleep pants" then the site should take them to a page named "Women sleep pants".


Some other navigation conventions:

  • Highlight my current location in navigation bars, lists or menus appear on the page.

  • Display Breadcrumbs at the top of the site, and use > between levels.

    For example: You are here: Home > Arts > Visual Arts > Nature and wildlife

  • Use tiny type to make breadcrumbs clear. Make the last item in the breadcrumbs bold.

  • The 4 reasons to use tabs are:

    • They're self-evident because everyone understands them.

    • They're hard to overlook and hard to mistake for anything but navigation.

    • They're handy.

    • They suggest physical space by creating the illusion that the active tab actually moves to the front.

  • The selected tab needs to be a different color in order to make it "pop" to the front.
